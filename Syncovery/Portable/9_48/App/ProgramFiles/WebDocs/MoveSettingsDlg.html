Move Settings


   These are special settings that apply only to MOVED FILES mode    

   If you choose to have destination files overwritten, you can    
   combine this setting with "Keep multiple backup versions" or    
   "Use Recycle Bin" for overwritten files.    
 

If a file already exists on the destination side:
Leave the files untouched
Overwrite destination file if identical or older
Rename the source file before moving it
Delete identical or older files from source

Move by Copying, then Deleting Source File
(This helps inheriting the permissions from the destination folder).
