Compressed Package Configuration

Instead of copying or zipping each file individually, you can create compressed
archives containing many files in each archive. Since the contents of
these backup archives is not visible to future synchronization runs, use
the Archive flag to determine which files need to be backed up.

This fearure can also be used to speed up sending many small files over a
network by packaging them in compressed archives. Use the Syncovery Remote
Service on the destination computer to unpack the files. This service must be
configured using the Syncovery Remote Service Control Panel.
